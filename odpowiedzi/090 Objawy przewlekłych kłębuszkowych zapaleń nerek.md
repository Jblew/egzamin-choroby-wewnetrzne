# 090 Objawy przewlekłych kłębuszkowych zapaleń nerek

## Final

**Podział:**

1. GPKZN
2. Ostre KZN
3. Przewlekłe KZN
   1. Mezangialne (najcz. nefropatia IgA)
   2. Submikroskopowe (najczęstsze u dzieci)
   3. Błoniasto-rozplemowe KZN (wtórne, najcz dp HCV)
   4. Błoniaste KZN (najczęstsza przyczyna zesp. nerczycowego u dorosłych; PF: kompleksy imunologiczne uszkadzają GBM)
   5. FSGS (ogniskowo-segmentalne stwardnienie kłębuszków) 



**Obrazy kliniczne:**

- Z. nerczycowy => poniżej
- Z. nefrotyczny => poniżej
- Postępująca PChN => [088 Leczenie przewlekłej choroby nerek- zasady ogólne](./odpowiedzi/088%20Leczenie%20przewlekłej%20choroby%20nerek-%20zasady%20ogólne.md)

**Charakterystyka przewlekłych KZN:**

1. **Mezangialne KZN** = Nefropatia IgA pierwotna lub wtórna
   1. PF: kompleksy IgA odkładają się w mezangium => uogólniony rozplem mezangium => duża ilość macierzy
   2. Epi: najczęstsze KZN u dorosłych
   3. Objawy: dominuje krwinkomocz + umiarkowany białkomocz
   4. Przyczyny: Nefropatia IgA pierwotna lub wtórna (marskość, celiakia, Schoenline-Henoch)
   5. Leczenie ACEI + GKS (w zależności od szybkości spadku GFR)
   6. Rokowanie: powolny postę, u 20% chorych po 20 latach schyłkowa PChN
2. **Błoniaste KZN** (najczęstsza przyczyna zesp. nerczycowego u dorosłych)
   1. PF: kompleksy imunologiczne uszkadzają GBM
   2. Epi: M, 40-60rż
   3. Objawy: z. nerczycowy + krwinkomocz. NT późno
   4. 80% Pierwotne: naucz p/fosfolipazie A2
   5. 20% wtórne: lite nowotwory (gł. płuca), HBV
   6. Leczenie: ACEI, immunosupresja tylko w cięzkiej postaci
   7. Rokowanie: samoistne remisje
3. **Submikroskopowe KZN** (= nefropatia zmian minimalnych)
   1. PF: jakiś nieznany czynnik we krwi zmniejsza ujemność ładunku GBM
   2. Epi: najczęstsze KZN u dzieci (tak ponad 80-90%), 25% z. nerczycowych u dorosłych
   3. Objawy: 
      1. z. nerczycowy często >10g/d szybko postępujący
      2. Prawidłowe BP
      3. Prawidłowy GFR
   4. Rozpoznanie z wykluczenia innych KZN
   5. Leczenie: 
      1. należy jak najszybciej osiągnąć remisję, bo tak duży białkomocz szybko daje powikłania z innych narządów 
      2. Prednizon => cyklofosfamid => takrolimus
   6. Rokowanie: pomyślne. Mała częstość nawrotów, są samoistne remisje.
4. **Błoniasto-rozplemowe KZN** (wtórne, najcz. do HCV)
   1. PF: rozlana proliferacja mezangium z pogrubieniem włośniczek
   2. Przyczyny: najcz. HCV, też postać pierwotna (gorzej rokuje)
   3. Objawy: zespół nefrytyczny (80% NT, białkomocz subnerczycowy, wałeczki ziarniste)
   4. Leczenie: eliminacja cz. ryzyka AKI i PChN. Immunosupresja dopiero jeśli białkomocz nerczycowy
   5. Rokowanie: niepomyślne. Po 5 latach 50% dializowanych
5. **FSGS KZN** (ogniskowo-segmentalne stwardnienie kłębuszków) 
   1. PF: 
      1. we krwi czynnik uszkdadzający podocyty => uszkodzenie podocytów => stwardnienie kłębuszków
      2. Ogniskowe, bo najpierw szkli wieją tylko kłębuszki w okolicy rdzenia
      3. Segmentowe, bo szkliwieją tylko fragmenty kłębuszków
   2. Epi: młodzi mężczyźni
   3. Objawy: najcz. z. nerczycowy, u 40% krwinkomocz, NT
   4. Przyczyny:
      1. Pierwotny
      2. Wtórne: stany hiperfiltracji
   5. Leczenie: prednizon
   6. Rokowanie: bardzo szybko niewydolność nerek i dializa

- Ostre poinfekcyjne KZN: z. nefrytyczny
