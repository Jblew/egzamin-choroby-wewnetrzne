# 069 Postępowanie w wyrównanej marskości wątroby

## Final

**Definicja:** 

- Uogólnione uszkodzenie miąższu
- => zmiana jednorodnej we włóknienie + guzki regeneracyjne 
- => zab. czynności wątroby
- => zab. struktury naczyniowej => nadciśnienie wrotne (wzrost ciśnienia w żyle wrotnej >12 mm Hg; prawidłowo ≤10 mm Hg)

**Przyczyny:** 

- alkoholowa choroba wątroby
- WZW typu B, D lub C,
- autoimmunologiczne zapalenie wątroby, 
- choroby metaboliczne (hemochromatoza dziedziczna i inne stany przeładowania żelazem, choroba Wilsona, niedobór α1-antytrypsyny
- mukowiscydoza
- NAFL/NASH
- choroby dróg żółciowych (niedrożność zewnątrzwątrobowych dróg żółciowych, niedrożność wewnątrzwątrobowych dróg żółciowych, pierwotne zapalenie dróg żółciowych, pierwotne stwardniające zapalenie dróg żółciowych)
- utrudnienie odpływu żylnego (choroba zarostowa żył wątrobowych, zespół Budda i Chiariego, prawokomorowa niewydolność serca)
- leki: metotreksat, metylodopa, amiodaron
- toksyny
- jelitowe zespolenie omijające (w leczeniu otyłości)
- marskość kryptogenna (przyczyna nieznana)

**Rozpoznanie:**

- jednoznaczne: biopsja => histopat
- zwykle wystarczą bad. obrazowe + obj. kliniczne + lab. + elastografia (nieinwazyjna ocena włóknienia)

**Leczenie:**

1. Bezwzględne zero alkoholu i zero tytoniu
2. Dieta:
   1. Dużo energii i białka
   2. Rozłożona na 6 posiłków, szczególnie ważna jest przekąska lekka przed snem z węglowodanów, która zapobiega nocnej glukoneogenezie z rozpadu białek i niedożywieniu białkowemu
   3. Leki hepatoprotekcyjne ani metionina nie są polecane (ew sylimaryna w dużych dawkach)
   4. Koniecznie suplementacja wit D
3. Aktywność fizyczna p/działa utracie masy mięśniowej
4. Leczenie przyczynowe: wg etiologii

**2. Leczenie przyczynowe** w zależności od etiologii marskości.

**3. Leczenie objawowe:**

1. Hiponatremia z hiperwolemią (= przewodnienie): ogranicz podaż płynów poniżej 125, w objawowej lub <110 suplementacja sodu
2. Hiponatremia z hipowolemią => przetocz 0,9% NaCl i lecz przyczynę (najcz odstawić diuretyki)
3. Zab. krzepnięcia — w większości nie należy podawać FFP jeśli chory nie krwawi albo krwawienie wynika z marskości (np. żylaki przełyku). Jeśli jest ostra zakrzepica to HDCz, ostatecznie warfaryna. Nie zaleca się NOAC
4. Hiperglikemia/cukrzyca — zwykle wystarczy dieta i nie potrzeba insulinoterapii
5. Leczenie powikłań: [068 Powikłania marskości wątroby - postępowanie](./068%20Powikłania%20marskości%20wątroby%20-%20postępowanie.md)
6. Inne nieswoiste leczenie
   1. Statyny zmniejszają nadciśnienie wrotne, szczególnie simwastatyna 20-40mg/d
   2. Szczepienie na WZWA, WZWB, grypę, pneumokoki
   3. Antybiotyki jako prewencja translokacji bakterii z PP
   4. B-blokery nieselektywne jako prewencja krwawienia z żylaków przełyku

7. Niewyrównana marskość => przeszczep wątroby
